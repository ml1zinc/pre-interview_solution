### Simple ToDo cli application

#### Usage: 
```bash
solution.py [OPTIONS] [OPT_ARGUMENTS]
```

#### Aviable options:
```
  -h, --help                show this help message and exit
  -a ADD [ADD ...],         Adding one or more ToDo tasks. Type every task in " "
  -l, --list                Get list of ToDo tasks
  -la, --all_list           Get list of DONE ToDo tasks
  -ld, --done_list          Get list of DONE ToDo tasks
  -ls, --done_statstic      Get statistic of DONE ToDo tasks
  -r REMOVE [REMOVE ...]    Remove one or more ToDo tasks. Type id`s of tasks
  -d DONE [DONE ...],       Mark one or more ToDo tasks are done. Type id of task
```
